🔗 Configuração da API (Google Apps Script)

Cole aqui a URL do seu Web App do Google Apps Script após fazer o deploy.

📋 Como configurar passo a passo

1. Abra o Google Sheets e vá em Extensões → Apps Script

2. Cole o conteúdo do arquivo google-apps-script.js

3. Execute a função setupDatabase() para criar todas as tabelas

4. Vá em Implantar → Nova implantação → App da Web

5. Configure: Executar como Eu, Quem acessa Qualquer pessoa

6. Copie a URL e cole no campo acima

7. Cole a mesma URL no arquivo do App do Veterinário
